Diverses

Release Candidates

Wir stellen auch sogenannte Release Candidates zur Verfügung. Diese Versionen sind die Offiziellen Releases plus alle Fehlerkorrekturen seit dem letzten offiziellen Release. Diese Installationsdateien werden nicht jeden Tag erstellt, sondern nur wenn notwendig. Sie können diese bei sich installieren wenn Sie nicht auf den nächsten offiziellen Release warten möchten oder können und Sie bereits wissen dass ein für Sie relevanter Fehler bereits behoben wurde.

Nightly Builds

Nightly Builds sind ebenfalls erhätlich. Diese werden jede Nacht automatisch erstellt und enthalten immer die aktuellste Entwicklerversion. Diese Installationsdateien sind nicht für den täglich Einsatz freigegeben sondern nur für Tests. Es kann auch vorkommen dass ein solcher Nightly Build gar nicht funktioniert. Installieren Sie diese nur auf eigenes Risiko!

Ältere Releases

Ältere Releases sind erhältlich auf der SourceForge files Seite.

Sourcecode

TortoiseSVN ist lizensiert unter der GPL. Das bedeutet dass der ganze Sourcecode ebenfalls frei verfügbar ist und Sie das Programm auch selber kompilieren können (sofern Sie das möchten).

Der Sourcecode liegt bei SourceForge.net in unserem Subversion Repository. Sie können mit Ihrem bevorzugten Webbrowser den Sourcecode hier ansehen.
