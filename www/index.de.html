TortoiseSVN 1.8.1 released

geschrieben am 23. Juli 2013

Wir sind stolz den Release von TortoiseSVN 1.8.1 zu verkünden. Der Release ist verlinkt mit Subversion 1.8.1

Dies ist ein Bugfix release ohne neue Features.

Für die wichtigsten Änderungen in Version 1.8 lesen Sie bitte die Release Notes (Englisch).

Bitte lesen Sie die Release Notes genau durch bevor Sie TortoiseSVN 1.8 installieren, um mögliche Überraschungen zu vermeiden.
