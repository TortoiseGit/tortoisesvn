TortoiseSVN is a Subversion client implemented as a windows shell extension.

For more information have a look at the Screenshots and the daily use guide. The impatient among you may download the latest installer from the file sharing area. Please read the FAQ before posting questions to the mailing list.

We also have a Guide on how to install a subversion server with apache.

Vote for TortoiseSVN on O'Reilly's OSDir.com

Please, please, please! If you have any request, ideas for new functions and features or just bug reports then mail them to the dev mailing list before filing an issue. It's annoying to ask further questions via issuezilla. Also read our rules for the list.

Mission

  • To provide a seamless Windows Explorer experience for working with Subversion controlled files.
  • If you have any questions please have a look at the faq and/or ask in the mailing list dev@tortoisesvn.tigris.org.

System Requirements

  • Windows 98/Me or Win2k SP2 or WinXP, IE5.5 or later
  • It also works on WinNT4 and Win95, but you need to have the IE4 shell extensions installed to get a newer shell version. The minimum shell version is 4.72! We have reports that TortoiseSVN works on these old systems, but use it at your own risk.

Implemented Features

  • Subversion file status is viewable directly in Windows Explorer
  • Advanced info on files viewable in the explorer property pages
  • Copy, Rename, Move of versioned files possible directly with explorer
  • Subversion operations are accessible from the Windows context menu (right click menu)

Planned Features

  • Planned: Repository browser for copy, move and merge operations
