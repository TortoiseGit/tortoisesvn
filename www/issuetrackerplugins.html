Issue tracker plugins

$ADSENSE_TOP$

TortoiseSVN provides an interface to enable plugins to integrate with issue trackers. There are plugins available for several bug/issue tracking systems.

Please note that this list isn't complete. If you know of a plugin that's not listed here, please send a mail to our mailing list with information about the plugin and where to find it.

These plugins are all separate projects and not related to TortoiseSVN in any way. We don't provide support for them or are responsible for whether they're working properly or not. If you need support with those, please ask on the corresponding project page for help.
