Donate to the TortoiseSVN developers

If you like to contribute to TortoiseSVN too or like to make the developer(s) happy then you can send us some presents.

While working on TortoiseSVN we love to listen to music. And since we work many hours on the project we need a lot of music. Therefore we have set up some wish-lists with our favourite music CD's.
And sometimes we need a break to gather our thoughts, that's why the wish-lists also have some DVD's in it. And if we really manage to switch off the PC, we might even grab a book and start to read :-)

Since we like a pretty layout, we created a Donation Matrix :-)

In case you don't know which one of us you want to donate, you can check out the commit statistics to get an idea how much work each of us does.

If you think that sending money is boring, because it doesn't have the personal touch that sending presents has, Lübbe can tell you what he'll do with the money. Get a Bass!
