Compatibility

The following matrix shows which versions of Subversion clients and servers can work together:

Compatibility

Client 1.0.x Client 1.1.x Client 1.2.x Client 1.3.x
Server 1.0.x yes yes (1) yes (1) yes (1)
Server 1.1.x yes yes yes(1) yes(1)
Server 1.2.x yes yes yes yes(1)
Server 1.3.x yes yes yes yes



(1) There may be some new functions which the server doesn't have yet but the client has. Then the server will return an error message telling you that the function isn't available.


Note: a repository dump/load cycle on the server won't be necessary until version 2.0.x comes out!
